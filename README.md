# MERN Todo App

[View demo](https://todo.savayer.space/)

The server [up here](https://todoserver.savayer.space/api/todos)

## Installation
Clone the repo and install the dependencies:

```
npm i
```

Then install the dependencies for the server:

```
cd server
npm i
```

Run the server
```
npm run server
```

Run the frontend
```
cd ..
npm run start
```
